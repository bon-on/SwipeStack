## English

- App Name: Swipe Stack
- Subtitle: Swipe, line up, and release into a sharper stack
- Short Description: Swipe to correct moving blocks, release to drop, and keep each overlap alive.
- Promotional Text: Make the last-second correction, release cleanly, and chase a taller run.
- Keywords: arcade, timing, stack, casual, offline, single-player, score
- Full Description:

Swipe Stack is a mobile timing game about swipe corrections, shrinking margins, and clean reruns.

Boxes glide across the top of the stage while you drag to nudge their path. Release to drop with a small carry of momentum, then only the overlapping section survives. Each successful stack makes the next one tighter and more demanding. Miss the overlap threshold and the run ends immediately. Your best stack is saved automatically so every attempt has a clear target.

Features:
- Swipe-to-align and release-to-drop controls
- Deterministic overlap-based stacking
- Landing guide and alignment feedback
- Speed increases every five successful stacks
- Crisp audio feedback for drops, saves, and failures
- Offline single-player play
